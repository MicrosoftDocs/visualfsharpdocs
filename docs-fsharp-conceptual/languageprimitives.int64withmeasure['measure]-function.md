# LanguagePrimitives.Int64WithMeasure<'Measure> Function

```fsharp
// Signature:
Int64WithMeasure : int64 -> int64<'u>
```

#### Parameters
*value*
Type: [int64](https://msdn.microsoft.com/library/1bec11c0-45ac-469e-923b-22a1708c0701)
