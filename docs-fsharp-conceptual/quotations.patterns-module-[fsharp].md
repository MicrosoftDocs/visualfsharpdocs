# Quotations.Patterns Module (F#)

Contains a set of primitive F# active patterns to analyze F# expression objects.

**Namespace/Module Path:** Microsoft.FSharp.Quotations

**Assembly:** FSharp.Core (in FSharp.Core.dll)


## Syntax

```fsharp
module Patterns
```
